# Yonoma Java SDK


## Installation

To install the Java SDK, add the following dependency to your project:

Gradle

```gradle
implementation 'io.yonoma:yonoma-java:1.0.0"'
```

Maven

```Maven
<dependency>
    <groupId>io.yonoma</groupId>
    <artifactId>yonoma-java</artifactId>
    <version>1.0.0"</version>
</dependency>

```
